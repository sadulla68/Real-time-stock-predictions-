# Real-Time Stock Price Prediction

This project predicts stock prices using LSTM and technical indicators.

## Tech Stack
- Python
- TensorFlow
- FastAPI
- yFinance
- Scikit-learn

## Features
- RSI Indicator
- Moving Averages
- LSTM model
- REST API prediction endpoint
## Project Workflow

1. Download stock data using yFinance
2. Apply preprocessing and scaling
3. Train LSTM deep learning model
4. Expose prediction API using FastAPI

## Future Improvements

- Add technical indicators (RSI, MACD)
- Add backtesting engine
- Deploy on cloud
- Add live dashboard
